# 요구사항 — 어드민 조직도 관리

## 기능 요구사항

### 회사 설정
- 회사명, 설명 입력/수정
- 조직도 최상단에 표시

### 부서 관리
- 부서 생성(이름, 설명, 상위부서 선택)
- 부서 수정, 삭제 (하위부서 있으면 삭제 불가)
- ↑↓ 버튼으로 동일 레벨 내 순서 변경
- 상위부서 변경으로 계층 이동
- 트리 형태 시각화 (들여쓰기)

### 사용자 할당
- 각 부서에 profiles 테이블 사용자 할당
- 이미 다른 부서에 속한 사용자도 중복 할당 가능
- 부서에서 사용자 제거 가능
- 사용자 이름/이메일 검색으로 추가

### 일반 사용자 조직도 뷰
- /org 페이지: 읽기 전용 트리
- 회사→부서→멤버 계층 표시
- 로그인 필요

## 비기능 요구사항
- RLS: 관리자만 쓰기, 팀 전체 읽기
- 모바일 반응형 (테이블 카드 레이아웃)
