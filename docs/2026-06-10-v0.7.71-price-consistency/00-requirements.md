# 00 Requirements — 가격 정합성 4결손
## 배경
DC-ANA 진단으로 4결손 확인: 전모델 사다리 미백필(74/77 partial), 콕핏·펼침 전파분 누락(x8 원가없음), 기준선택↔전략가 갭, A100 x8 중복.
## 기능 요구
- FR1 전 모델 x1/2/4/8 행 존재(백필). 견적 없으면 "견적 대기".
- FR2 콕핏 원가/가격표 펼침에 전파(추정) 원가 표시 + "추정" 배지. _derived 행 펼침 허용.
- FR3 기준선택(원가)↔우리판매가(전략가) 관계 UI 안내 + 판매가후보 [지정].
- FR4 A100 x8 중복 정본화.
## 비기능
- 가격 SSOT(buildCatalog) 불변. 전파=추정 명시. 토큰/공용클래스/table-card. 멱등 마이그. push 금지.
## 제외
- 가격 계산식 변경, 신규 가격 개념.
