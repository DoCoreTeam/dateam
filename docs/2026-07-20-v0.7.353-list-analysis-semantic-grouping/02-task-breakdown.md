## 의존 관계

```
Phase 0 ──┬─→ Phase 1 ──→ Phase 2 ──┬─→ Phase 3
          │                          └─→ Phase 4 ──→ Phase 5
          └─ (T0.3 감사는 Phase 4 T4.1 전에 반드시 완료)
```

- Phase 1은 AI 없이 진행 가능 — Phase 0과 병행 가능
- Phase 3과 Phase 4는 Phase 2 완료 후 병렬 가능
- **T4.1(템플릿 이식)은 T0.3(마이그 감사) 완료 전 착수 금지** — 번호 충돌 재발 방지
