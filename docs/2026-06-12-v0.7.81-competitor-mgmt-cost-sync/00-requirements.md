# 00 요구사항 (v0.7.81)
## 사용자 요구
1. 용어 공통화: 라벨을 그때그때 박지 말고 B2B 한국 시스템 표준 용어 SSOT로 통일.
2. 경쟁사 관리 탭(공급사 탭 대칭): 목록/CRUD/검색/일괄삭제/일괄 공급사지정. 모델 하나하나 안 들어가게.
3. 항목별 "원가 인입" 폐기 → 저장된 출처(URL)로 재수집해 공급가 자동 반영(1버튼).
4. 실견적 우선: 실제 공급사 견적 > 경쟁가 추종가.
5. 값 변경 시 무조건 검토대기(승인 전 미반영).
6. 일괄적용=일괄삭제 대칭(CRUD 핵심).
## 비기능
용어 SSOT, 디자인토큰/공용컴포넌트, table-card 반응형, RLS·admin, 소프트삭제, 감사로그, git push 금지.
