## A. 기능 (MVP)
- [ ] 부서장/부서원이 부서 업무를 제목·설명·담당자·마감·우선순위로 등록
- [ ] 담당자 지정/변경은 **부서장·admin만**(전용 assignTask), 부서원 등록 시 담당자=본인/비움 (D-3)
- [ ] 상태(예정/진행/완료/보류) + 진행률(0–100) 갱신
- [ ] 체크리스트(선택) 추가·체크
- [ ] 댓글로 진행상황 기록·상호 열람(작성자 식별)
- [ ] 부서업무 리스트 상태별 필터로 진행 현황 한눈 확인
- [ ] 주간보고에서 부서업무 자동 집계 표시 + 수동 인용(읽기 전용)

## B. 권한/보안 (게이트 — 하나라도 ❌면 미완)
- [ ] RLS 매트릭스(부서장/부서원/타부서/임원/admin) 전부 의도대로
- [ ] `task_kind='personal'` 개인 일일업무 **본인만 접근** 회귀 없음
- [ ] 계층플래그 OFF 시 본인/관할로 축소(ON/OFF 일관)
- [ ] 댓글: 부서 가시자만 읽기, 본인 댓글만 수정/삭제
- [ ] 🟥 DC-SEC 리뷰 CRITICAL/HIGH 0

## C. 재사용 원칙 준수 (사용자 지시)
- [ ] 신규 권한 함수 0 — 기존 `private.my_*`/`is_executive`/`hierarchy_enabled` 재사용
- [ ] org-scope·DailyTaskSelector·SWR·entry_type/priority/target_date 재사용 확인
- [ ] 신규 테이블 최소(ⓐ안=0개, ⓑ안=1개 이내)
- [ ] 일일업무 화면과 부서업무 화면 분리(개념 혼동 없음)
