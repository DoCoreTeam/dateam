# Requirements — newAX v0.2.0 Supabase Platform

## 배경
기존 localStorage 기반 정적 HTML 대시보드를 팀 전체가 사용하는 협업 플랫폼으로 전환.

## 기능 요구사항

### 인증
- 이메일 + 비밀번호 로그인
- 수동 로그아웃 전까지 세션 유지 (persistSession)
- 어드민 role / 일반 팀원 role 구분

### 팀원 화면
- 기존 대시보드 (조직도·KPI카드·운영루틴) 이식
- 일별 루틴 체크박스 (월~일, 주 단위: 월 00:00 ~ 일 23:59)
- KPI 수치 입력 (metric명, 수치, 단위, 기간)
- 주간보고 작성 (구분·성과·계획·이슈/협조사항)
- 본인 작성 이력 조회·편집

### 어드민 화면
- 팀원 계정 생성·비활성화
- 전체 팀원 주간보고 취합 (주차 선택)
- 루틴 달성률 현황
- KPI 집계 현황

## 비기능 요구사항
- RLS로 본인 데이터만 CRUD, 어드민은 전체 READ
- Vercel 자동 배포 (GitHub push → 빌드)
- 기존 GitHub Pages HTML 파일 유지 (병행 운영)
