# Paper Workspace 디자인시스템 분석 & newAX 적용 기획

> 작업일: 2026-06-06 · v0.7.25 · 산출물: `test.html`(독립 프리뷰, 실서비스 비반영)

## 1. `design seti.html` 디자인시스템 분석

**컨셉:** "Paper Workspace" — 종이/모눈/데스크 메타포. 디지털 대시보드를 *책상 위에 놓인 종이 작업물*처럼 표현.

### 디자인 토큰
| 범주 | 토큰 | 값 | 의미 |
|------|------|----|----|
| 배경 | `--bg-canvas` | `#EAE6D9` | 크라프트지/책상 바탕 (따뜻한 베이지) |
| 배경 | `--bg-paper` | `#FFFFFF` | 흰 종이 (카드) |
| 텍스트 | `--text-primary` | `#2C2C2C` | 잉크 블랙 (순흑 아님) |
| 텍스트 | `--text-muted` | `#888888` | 흐린 잉크 |
| 라인 | `--grid-line` | `rgba(200,204,215,.4)` | 모눈종이 격자 |
| 강조 | `--tape-bg` | `#FDE047` | 노란 마스킹테이프/형광펜 |
| 강조 | `--marker-red` | `#E11D48` | 빨간 펜 |
| 강조 | `--accent-blue` | `#0369A1` | 파란 펜 |
| 모양 | `--grid-size` | `28px` | 격자 간격 |
| 모양 | `--radius-paper` | `6px` | 종이 모서리 |
| 폰트 | `--font-ui` | Inter | 본문/UI |
| 폰트 | `--font-marker` | Nanum Pen Script | 손글씨 마커 |

### 시그니처 요소 (이 디자인을 "이 디자인"으로 만드는 4가지)
1. **`.paper-stack`** — 6겹 box-shadow로 *여러 장 겹친 종이* 입체감. 핵심 식별자.
2. **`.grid-pattern`** — 모눈종이 격자 배경(28px). 카드 안쪽에 적용.
3. **`.tape-label`** — 마스킹테이프/포스트잇. `transform: rotate(-1deg)` + 불규칙 `border-radius`(손으로 찢은 느낌) + 손글씨 폰트.
4. **타임라인 피드** — 점(dot) + 세로 연결선(`::before`)으로 시간순 활동.

## 2. newAX 적용 전략

### 매핑 원칙
- newAX의 **정보 구조는 유지**, **표피(skin)만 종이 메타포로 치환**.
- 기존 인디고 계열 액센트(`#6366f1`) → 종이 액센트(노랑 테이프/빨강·파랑 펜)로 재해석.
- 손글씨 폰트는 **장식 텍스트(인사·메모·강조 라벨)에만** 제한 — 본문 가독성은 Inter 유지.

### seti → newAX 홈 대시보드 매핑
| seti 요소 | newAX 홈 대응 |
|-----------|---------------|
| sidebar nav | 홈/일일업무/주간보고/캘린더/KPI/루틴/본부운영 |
| "좋은 아침입니다 👋" + 테이프 | "안녕하세요, OOO님 👋" + 오늘 날짜 + 테이프 인사 |
| 진행 중인 프로젝트(progress) | 오늘 업무 체크리스트 + 이번 주 KPI 진행 바 |
| quick-add input | 일일업무 빠른 입력 |
| 최근 활동 피드(타임라인) | 최근 활동 피드 |
| 테이프 메모 | 확인 안 한 메모 / 주간보고 |
