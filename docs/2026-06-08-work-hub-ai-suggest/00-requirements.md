# 업무 허브 통합 + AI 업무 추출 — 요구사항 정의서 (기획)

> 2026-06-08 · 상태: **기획(구현 0)** · 대상 v0.8.0 · 본 문서는 합의용. 코드/마이그레이션/커밋 없음.

## 1. 문제 정의 (Why)
- **분산된 업무 메뉴**: 일일업무·부서업무·주간보고가 사이드바 3개로 흩어져(비연속 위치) 인지부하·중복 진입.
- **부서업무 수동 생성 노동**: 부서장이 부서원의 일일업무/주간보고를 일일이 읽고 손수 "부서업무"로 옮겨 적어야 함. AX본부 정체성("AI 다 담기")과 맞지 않음.

## 2. 두 기능 (What)
### ① AI 업무 추출/제안
일일업무(daily_logs) + 주간보고(weekly_reports) 데이터를 **AI가 분석해 "새 부서업무 후보"를 추출** → 사용자가 **제안 목록에서 골라(체크) 담당자·마감 조정 후 일괄 등록**.
### ② 업무 메뉴 통합
일일업무·부서업무·주간보고를 **단일 "업무" 메뉴 + 상위 탭 3개**로 통합.

## 3. 확정 결정 (Q&A 2026-06-08) ✅
- **D-1 AI 추출 범위** = org-scope: **부서장 → 관할 부서원 전체** / **일반 → 본인** (기존 `resolveOrgScope`/`deptMemberUserIds` 재사용).
- **D-2 결과 처리** = **제안 목록 체크 → 일괄 등록** (DailyTaskSelector 패턴 재사용). 자동 등록 금지.
- **D-3 메뉴 IA** = **단일 "업무" 메뉴 + 탭 3개**(일일/부서/주간).
- **D-4 부서업무 위상** = **AI 제안이 주경로 + 수동 등록(현 S1~S4) 보조 유지**. 완전 자동파생 아님.

## 4. 사용자 스토리 (MVP)
- US-1 부서장으로서, "AI로 부서업무 후보 찾기"를 누르면 최근 부서원 일일/주간에서 후보가 제안된다.
- US-2 제안 목록에서 원하는 후보를 체크하고 담당자·마감을 조정해 한 번에 부서업무로 등록한다.
- US-3 이미 비슷한 부서업무가 있으면 "중복 의심" 표시로 알려준다(자동 머지 X).
- US-4 일반 사용자로서, 내 최근 일일/주간에서 내 할 일 후보를 받아 본인 업무로 등록한다.
- US-5 사이드바 "업무" 하나로 들어가 탭으로 일일/부서/주간을 오간다.

## 5. 범위 (MVP vs 후속) — 🟦 DC-BIZ 권고
| 영역 | MVP | 후속 |
|------|-----|------|
| AI 트리거 | **수동 버튼만** | 자동 스케줄(주1회), 작성시 인라인 제안 |
| AI 범위 | 부서장=부서원전체 / 본인=본인, 최근 2주 | 기간 확장, 임베딩 군집 |
| 결과 | 체크→일괄등록 + 중복의심 표시 | 채택률 피드백 튜닝, 알림 |
| 비용 가드 | 기간 캡(2주, 최대 4주)·상태필터(doing/planned/blocker)·1일 호출 상한 | 4주↑ 시 주간요약 대체 |
| IA | 단일 /work + 탭3 + 기존 URL 301 | 탭 간 승격 버튼, 통합 검색/필터 |

## 6. 비범위
- 모든 코드 구현·마이그레이션·커밋. AI 자동등록. 임베딩 신규 적재(일반 daily_logs엔 현재 embedding 없음 — DC-ANA 확인).

## 7. 성공지표 (측정 가능 — DC-BIZ)
1. 부서업무 수동 생성 평균 시간 −50%
2. AI 후보 채택률 ≥ 40% (미만이면 프롬프트 재튜닝 트리거)
3. 통합 "업무" 메뉴 진입수 ≥ 기존 3메뉴 합산의 90%(이탈 없음)
- 보조: 주간 Gemini 호출 비용(폭주 감지)

## 8. 핵심 제약 (가드 — DC-BIZ 1순위)
- **AI 비용**: 호출은 수동 트리거만, 기간/건수 캡, `logTokenUsage` 추적.
- **환각/오추출**: source_quote 근거 강제(없으면 후보 제외), confidence<0.7 제외, "후보 제안+사용자 선택"으로 안전.
- **중복**: 부서+제목 유사도로 "이미 있음" 표시(자동 머지 금지).
- **IA 통합**: 기존 `/daily`·`/dept-tasks`·`/weekly-report` URL **301 리다이렉트** 필수(북마크 보존).
- 개인 일일업무 격리(task_kind='personal') 무회귀, org-scope 권한 재사용.

## 9. 권고 순서 (DC-BIZ)
② 메뉴 통합(그릇) → ① AI 추출(부서업무 탭의 "AI 후보" 섹션으로 얹기). 문서는 통합 1건.
