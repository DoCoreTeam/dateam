# AI 스마트 저장 — 요구사항 정의

## 배경
/daily 페이지의 기존 "저장" 버튼을 AI 처리 기반으로 교체.
사용자가 자유형 텍스트로 업무를 입력하면 AI가 맥락을 파악하여
분류 · 분리 · 구조화 · 연결 제안을 수행한 뒤 저장.

## 핵심 기능 요구사항

### F-01. 실시간 분석 미리보기
- 입력창에 텍스트 입력 중 debounce(800ms) 후 AI가 백그라운드에서 분석
- 입력창 하단에 "AI가 N개 항목을 감지했어요" 힌트 표시
- 전체 결과는 AI 저장 클릭 시에만 패널/모달로 표시

### F-02. AI 저장 버튼 (기존 저장 완전 교체)
- 버튼 레이블: "AI 저장" (기존 "저장" 제거)
- 클릭 시 AI API 호출 → 스트리밍으로 결과 패널에 출력
- Ctrl+Enter 단축키 동일하게 적용

### F-03. 결과 확인 패널 (슬라이드 or 모달 — 구현 시 결정)
- AI 처리 결과를 카드 형태로 표시
- 각 항목: 제목 / 상태 / 날짜·시간 / 우선순위 / 연관 거래처·담당자
- 직접 편집 가능 (인라인 편집)
- "다시 분석" 버튼으로 재처리 가능
- "확정 저장" 버튼으로 최종 저장

### F-04. AI 추출 범위
| 추출 항목 | 설명 | 예시 입력 → 출력 |
|----------|------|-----------------|
| 상태 분류 | 완료/진행중/예정/블로커/메모 | "오전에 미팅 완료" → 완료 |
| 다중 항목 분리 | 여러 업무를 각 항목으로 분리 | "미팅하고 기획서 작성" → 2개 |
| 날짜·시간 추출 | 상대/절대 표현 모두 파싱 | "내일 오전 10시" → 2026-05-28 10:00 |
| 우선순위 감지 | 긴급/높음/보통/낮음 | "긴급하게 처리해야 함" → 긴급 |
| 거래처·담당자 태깅 | 기존 DB 데이터와 매칭 | "A사 김부장" → 거래처:A사, 담당자:김부장 |

### F-05. 연동 제안 (체크박스)
결과 패널 하단에 선택적 연동 옵션 표시:
- [ ] 캘린더에 일정 추가 (날짜 감지된 항목만 표시)
- [ ] 주간보고에 포함 표시
- [ ] 루틴 체크에 연결

### F-06. 스트리밍 출력
- 결과 패널에 AI 응답이 타이핑되듯 자연스럽게 나타남
- 항목이 하나씩 순차적으로 나타나는 애니메이션

### F-07. 주간보고 AI 초안 생성
- 주간보고(/weekly) 페이지에 "AI 초안 생성" 버튼 추가
- 이번 주 일간 업무 로그 전체를 컨텍스트로 전달
- 업무 요약 + 주요 성과 + 다음 주 예정 구조로 초안 생성

## 비기능 요구사항
- AI API 응답 목표: 3초 이내 첫 토큰
- 오프라인/API 실패 시: 기존 저장 방식으로 폴백 (에러 토스트 + 일반 저장)
- AI 모델: 시스템 설정에서 지정된 모델 사용 (하드코딩 금지)
