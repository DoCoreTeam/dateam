# AI 스마트 저장 — UX 플로우

## 화면 상태 다이어그램

```
[초기 상태]
┌─────────────────────────────────────────┐
│ 완료  진행중  예정  블로커  메모          │
│                                         │
│ ┌─────────────────────────┐  ┌────────┐ │
│ │ 업무 내용 입력...        │  │ AI 저장 │ │
│ │                         │  │ Ctrl+↵ │ │
│ └─────────────────────────┘  └────────┘ │
└─────────────────────────────────────────┘

     ▼ 입력 시작 (debounce 800ms)

[분석 힌트 표시]
┌─────────────────────────────────────────┐
│ ┌─────────────────────────┐  ┌────────┐ │
│ │ 오전에 A사 미팅 완료.   │  │ AI 저장 │ │
│ │ 내일 기획서 제출 예정.  │  └────────┘ │
│ └─────────────────────────┘             │
│ ✨ 2개 항목 감지됨                       │
└─────────────────────────────────────────┘

     ▼ "AI 저장" 클릭

[결과 패널 슬라이드 (스트리밍)]
┌──────────────────────────────────────────────────────┐
│ 입력창 영역          │  AI 분석 결과                  │
│                      │  ─────────────────────────    │
│                      │  ┌─────────────────────────┐  │
│                      │  │ 📋 A사 미팅              │  │
│                      │  │ 상태: 완료  우선순위: 보통 │  │
│                      │  │ 거래처: A사               │  │
│                      │  └─────────────────────────┘  │
│                      │  ┌─────────────────────────┐  │
│                      │  │ 📋 기획서 제출            │  │  ← 스트리밍으로 나타남
│                      │  │ 상태: 예정  우선순위: 보통 │  │
│                      │  │ 일정: 2026-05-28          │  │
│                      │  └─────────────────────────┘  │
│                      │                               │
│                      │  연동 제안:                    │
│                      │  ☑ 캘린더에 일정 추가         │
│                      │  ☐ 주간보고에 포함            │
│                      │                               │
│                      │  [다시 분석]  [확정 저장]      │
└──────────────────────────────────────────────────────┘
```

## 인터랙션 세부 사항

### 실시간 힌트 (F-01)
- 입력 없을 때: 힌트 없음
- 1개 감지: "✨ 1개 항목 감지됨"
- 복수 감지: "✨ N개 항목 감지됨"
- 힌트는 텍스트만, 클릭해도 아무 동작 없음 (AI 저장 버튼 유도)

### 스트리밍 애니메이션 (F-06)
- 패널이 오른쪽에서 슬라이드 인
- 첫 카드부터 순서대로 fade-in + 아래로 슬라이드
- 텍스트는 타이핑 효과 없이 카드 단위로 등장 (깔끔하게)
- 로딩 중에는 skeleton 카드 표시

### 카드 직접 편집 (F-03)
- 각 필드 클릭 시 인라인 편집 모드
- 상태: 드롭다운, 날짜: datepicker, 거래처·담당자: 기존 검색 컴포넌트 재사용
- 제목: 텍스트 인풋

### 오류 상태
- AI 처리 실패: 패널 대신 토스트 "AI 분석 실패. 일반 저장하시겠어요?" + [저장] [취소]
- 부분 실패 (일부 항목만 파싱): 파싱된 항목만 표시 + "일부를 파악하지 못했어요" 안내
