## 구현 완료 기준 (구현 단계 시)

### 기능 검증
- [ ] 자유형 텍스트 입력 → 다중 항목 분리 동작
- [ ] 상태(완료/진행중/예정/블로커/메모) 자동 분류 정확도 80%+
- [ ] 날짜·시간 추출 ("내일", "다음주 월요일", "오전 10시" 등)
- [ ] 거래처·담당자 이름 매칭 (완전 일치 + 부분 일치)
- [ ] 우선순위 감지 (긴급/높음 키워드)
- [ ] 직접 편집 후 저장 → DB 반영
- [ ] "다시 분석" 재처리 동작
- [ ] 캘린더 연동 체크박스 → 실제 캘린더 등록

### 오류 처리
- [ ] AI API 실패 시 폴백 토스트 + 일반 저장 동작
- [ ] 빈 입력 시 AI 저장 버튼 비활성화
