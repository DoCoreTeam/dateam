# 파트너 등급 및 할인율 관리 — 요구사항

버전: v0.5.1 | 날짜: 2026-06-01

## 목표
어드민이 파트너 등급(이름 자유 입력)과 각 등급의 할인율(%)을 CRUD할 수 있는 기능 구현.

## 기능 요구사항
1. 파트너 등급 목록 조회 (이름, 할인율, 생성일)
2. 신규 등급 생성 (이름, 할인율 0~100%)
3. 기존 등급 수정 (이름, 할인율)
4. 등급 삭제 (확인 후 삭제)
5. 어드민 전용 — admin role만 접근 가능

## 비기능 요구사항
- 반응형 UI (mobile-first)
- 브라우저 테스트 필수
- RLS 정책으로 admin만 쓰기, 팀원 전체 읽기
