# 요구사항 — 기준 공급가 채택 + gcube 공시가 분리 (v0.6.37)
## 문제
- gcube.ai 공시 '판매가'가 supply_quotes(공급원가)로 적재(027) → 타사 매입원가와 동일선상 비교 → 이중마진/최저가 왜곡.
- 가격표가 무조건 자동 최저가만 사용 → 실제 조달 가능한 견적을 사람이 고를 수 없음.
## 요구
1. gcube(list)를 공급원가 풀에서 제외, '시장 참고가'로 표시.
2. 상품별로 기준 공급가 '선택(채택)' — 선택분이 고객 판매가 기준.
3. 미선택 시 자동 최저가(회귀 없음), 선택 만료 시 자동 폴백+경고.
4. 가격표·고객판매가격표·시장비교 3화면 일관 반영, 선택 상태 명확 표시.
