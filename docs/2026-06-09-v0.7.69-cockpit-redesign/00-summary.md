# 콕핏 재설계 — 쉬운 말 + 기본기 + 클릭 상호작용 (v0.7.69)

## 작업
가격 결정 콕핏을 사용자 멘탈모델대로 재정리. 전문용어 제거, 검색/필터/정렬 추가, 셀 클릭 즉시 펼침 + 관장화면 이동.

## 컬럼 (어려운 말 전부 제거)
1. 모델·구성
2. gcube 사이트 가격 (옛 "공시가" — 변동값)
3. 원가 — 공급사 중 최저~최고 범위
4. 판매가 후보 — 최저원가 + 마진. [이 값으로 지정]→전략가 승격
5. 경쟁사 가격 — 경쟁사 중 최저~최고. 클릭→회사별 전환/비교
6. 우리 판매가 — 전략가(연필 편집), 최종 결정가
※ "시장중앙(median)/시장편차" 완전 삭제

## 기본기 (지금 없음 → 추가)
- 검색(모델명) / 필터(Tier·미설정) / 정렬(가격·마진) — 클라이언트(약 133행)
- 셀/컬럼 클릭 → 그 자리서 바로 펼침(별도 상세보기 버튼 제거 가능)
- 펼친 상세 항목 클릭 → 관장 화면 이동:
  · gcube 사이트 가격 → 가격표 탭(list)  · 원가 → 공급사/견적  · 경쟁사 → 시장비교 탭  · 우리판매가 → 변경이력
- 공급사 최저/최고, 경쟁사 최저/최고 명시(나머지 차이 지표 근거)

## 수정 파일
- apps/web/app/api/pricing/gpu/cockpit/route.ts (공급사·경쟁사 breakdown + min/max + gcube가격 + 갱신일)
- apps/web/app/(member)/pricing/gpu/tabs/PriceCockpitTab.tsx (컬럼/라벨/검색·필터·정렬/셀펼침/이동/경쟁사전환)
- globals.css (.cockpit-* 보강), 필요시 status-colors

## 이유
직전 콕핏이 검색·필터·정렬 없이, 막무가내 용어(시장중앙)로 만들어져 사용자가 판단 불가. 기본기와 쉬운 표현으로 재정비.

## 영향
콕핏 탭 + 콕핏 API. 가격 계산 SSOT(buildCatalog) 불변. 기존 탭 무관.

## 추가 확정 (사용자 최종 피드백)
- **전략가 = 시장 포지셔닝 가격**: "우리 판매가" 컬럼 라벨/툴팁을 "포지셔닝 가격(시장에서 우리 위치)"로 명확화. 시장 최저~최고 범위 옆에서 어디에 설지 정하는 값.
- **gcube 사이트 가격도 콕핏에서 직접 편집** (지금껏 수정 UI 없던 list 견적): gcube 사이트 가격 셀도 연필 인라인 편집 → list 견적(price_type='list') 값 수정. admin+audit+revalidate.
- 편집 지점이 2개가 됨: ① gcube 사이트 가격(대외 표시가) ② 우리 판매가=포지셔닝 가격(전략가). 둘은 다른 값(혼동 방지 라벨/색 구분).
