# RALPH v0.7.280 — 검토 CRUD 갭 + 회의노트 가독성

## 배경
v0.7.279에서 GPU 검토 held 인카드 조치를 구현하며 CRUD 갭 5건 식별. 사용자가 "갭 다 구현 + 회의노트 줄바꿈 가독성"을 /ceo-ralph로 지시(범위=전부, 마이그=운영 직접 적용).

## 작업 항목
- **G0 회의노트 요약 가독성**: 요약이 한 덩어리 나열식으로 렌더됨 → 문단/불릿 구조 보존 렌더(또는 요약 포맷 구조화).
- **G1 no_variant 딥링크 정확화**: "모델은 있으나 그 장수 구성 없음"인데 newModel(신규 모델 등록)로 보내 중복 모델 생성 위험 → 기존 모델에 '구성 추가' 딥링크.
- **G2 필드 수동 보정**: 검토 카드에서 price/model_name/pricing_model을 AI recheck 없이 인라인 편집 → override_extracted로 확정 반영.
- **G3 review_items 소프트삭제**: 하드 `.delete()` → `deleted_at` 마이그 + update 소프트삭제 + 조회 필터.
- **G4 bulk 인카드 조치 대칭**: 일괄 확정 실패 항목의 held 사유/후보를 결과에 포함해 개별 조치 유도.
