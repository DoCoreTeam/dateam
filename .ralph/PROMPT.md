# Ralph Loop — 루틴 빈도 재설계 + 주간보고 모달 에디터
생성일: 2026-05-21

## 목표
1. 루틴 항목 daily/weekly 빈도 지원 (기본: weekly)
2. 주간보고 텍스트 에디터 → 모달 방식 풀 에디터

## 완료 기준
1. [ ] 주간 루틴 → 주 1회 단일 체크박스, 체크/해제 정상
2. [ ] 주간보고 에디터 → 클릭 시 모달 오픈
3. [ ] tsc --noEmit 에러 없음
4. [ ] 어드민 루틴 집계 정확

## 제약사항
- DB 스키마 변경 없음
- 기존 string[] 포맷 하위호환 (string = weekly)
